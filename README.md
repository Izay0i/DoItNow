A simple mobile reminder application

![Logo](./assets/images/icon_1024.png)

# Features
+ Add, modify or delete a task
+ Location support
+ Language support
+ Dark theme

# Supported OS
+ Android

## How to run
+ Install Expo CLI:\
`npm install --global expo-cli`
+ Then in the project folder, install the packages:\
`npm install`
+ After that:\
`expo start`
