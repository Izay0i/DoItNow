# DO IT NOW
A simple reminder mobile app

![Logo](./assets/images/logo.png)

# Features
+ Add, modify and delete a task
+ Location support
+ Language support
+ Dark theme

# Supported OS
+ Android

## How to run
Supply your own `LOCATION_AUTOCOMPLETE_API` & `ACCESS_TOKEN` in `.env` file\
`npm install`\
`expo start`

### Made with
+ React Native
